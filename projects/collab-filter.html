Collaborative Filtering

colab paper

Overview

This project explores collaborative filtering techniques how they are used to build personalized recommendations. Using the MovieLens dataset, I compared different collaborative filtering methods such as item-based and user-based using cosine similarity and matrix factorization to evaluate their accuracy and performance. The goal was to understand how recommendation systems work behind the scenes and how mathematical models can be applied and scaled to handle larger-scale data.

Technical Details

  • Dataset: MovieLens 100k
  • Environment: Google Colab
  • Languages + Libraries:
    • Python
    • NumPy
    • MatPlotLib
    • scikit-learn

Challenges
