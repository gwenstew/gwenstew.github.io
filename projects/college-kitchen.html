Overview

College Kitchen is a full-stack react app designed to address a common problem in college students: cooking for themselves for the first time. Meal planning can feel overwhelming when juggling school, budget and little cooking experience. College Kitchen provides a way for students to plan and prepare meals. Users can create an account, track groceries in their virtual pantry, and discover recipes that make use of what they already have. This reduces food waste whilst helping students save money and gain experience in the kitchen.

Technical Details

College Kitchen is built with React and utilizes Google OAuth for secure authentication and Firebase for real-time database management. Recipe discovery is powered by Spoonacular API to fetch for recipes that include the ingredients from the virtual pantry.
